# Velora Venues

Velora Venues is a full-stack mobile application developed as a third-year B.Tech Computer Science mini project. The application simplifies the process of discovering and booking wedding halls and auditoriums through a centralized platform.

## Features

### User Features

* User Registration and Login
* Secure Authentication
* Browse Wedding Halls and Auditoriums
* Search and Advanced Filtering
* Location-Based Hall Discovery
* Hall Details with Images, Capacity, and Amenities
* Wishlist/Favorites Management
* Real-Time Booking System
* Booking History
* Ratings and Reviews
* Google Maps Integration
* Razorpay Payment Integration
* Invoice Generation

### Hall Owner Features

* Register and Manage Halls
* Update Hall Details
* Manage Booking Requests
* Track Reservations

### Admin Features

* Approve or Reject Hall Listings
* Manage Users and Owners
* Monitor Booking Activities
* Maintain Platform Integrity

## System Architecture

The application follows a three-tier architecture:

1. Presentation Layer (Flutter UI)
2. Business Logic Layer (Node.js + Express)
3. Data Layer (MongoDB)

## Objectives

* Simplify venue discovery and booking.
* Provide transparent pricing and availability.
* Prevent double bookings.
* Improve reservation management.
* Provide a user-friendly mobile platform.
